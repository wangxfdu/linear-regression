# linear-regression

## 环境搭建(仅需一次)
* 下载 Anacond https://repo.continuum.io/archive/Anaconda2-5.0.1-Windows-x86_64.exe
* 双击安装，安装路径可改
* 安装wxPython包
  * 在程序中找到"Anaconda Prompt"，点击运行
  * 输入conda install -c anaconda wxpython，回车。若有提示，输入y。（此步骤需要网络）
  * 安装成功，关闭退出
## 运行程序
  * 下载本工程中的main.py，保存到本地，如c:\test1\test2
  * 在程序中找到"Anaconda Prompt"，点击运行
  * cd c:\test1\test2
  * python main.py
